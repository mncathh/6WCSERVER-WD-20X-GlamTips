# Glam Tips – Nails and Eyelash Online Appointment and Management System

Project Type: Service Booking and Management System (MEVN Stack)

Description:
Glam Tips is a web application designed to make booking nail and eyelash appointments simple and convenient for clients while providing salon owners with efficient management tools. Clients can browse services, book appointments, and manage them (edit, delete, confirm, or review) without creating an account. Salon owners can track all bookings, add services, or cancel appointments through the Admin Dashboard, while all data is securely stored in MongoDB.

Key Pages and Their Functions:

Home Page: Welcomes visitors and highlights the salon’s services.

About Us Page: Shares the salon’s story, staff profiles, and background information.

Services Page: Lists all available nail and eyelash services with descriptions and pricing.

Booking Page: Allows clients to schedule appointments and manage existing bookings.

Contact Us Page: Provides business contact details, location, and operating hours.

Admin Page: Enables the salon owner to view, cancel, and manage appointments, as well as add new services.

Main Features:

For Clients: Browse services, book appointments, manage bookings (edit, delete, confirm, review).

For Admin/Owner: View all appointments, add services, cancel bookings, and monitor the salon’s schedule.
